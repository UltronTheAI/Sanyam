# 🔐 Security

Sanyam is a personal self-control app, not a bank vault guarded by dragons.

## Be Careful With

- Accessibility access
- VPN permission
- Usage access
- Emergency unlock code

## Report A Problem

Open an issue with:

- what happened
- Android version
- phone model
- steps to repeat it

Please do not post private data or screenshots with personal info. The internet has raccoons.
