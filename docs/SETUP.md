Sanyam needs a few Android powers. Not superhero powers. More like hall-monitor powers.

## 1. Usage Access

Lets Sanyam see screen-time stats.

Open the app → tap **Request usage access** → allow Sanyam.

## 2. Accessibility

Lets Sanyam send blocked apps back home during sleep mode.

Open the app → tap **Open accessibility setup** → enable Sanyam.

## 3. VPN

Lets Sanyam use family-safe DNS.

Open the app → tap **Prepare VPN blocker**.

For stronger blocking, turn on **Always-on VPN** in Android settings.

## 4. Emergency Code

Default:

```txt
1441
```

Do not forget it, sleepy warrior.
